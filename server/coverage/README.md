# Test Coverage Report

This directory contains the test coverage reports generated by Jest.

## Viewing Coverage

Open `index.html` in your browser to view the interactive coverage report:

```bash
# macOS
open coverage/index.html

# Windows
start coverage/index.html

# Linux
xdg-open coverage/index.html
```

## Coverage Files

- `index.html` - Main coverage report
- `lcov.info` - LCOV format for CI tools
- `coverage-summary.json` - JSON summary
- Various subdirectories with detailed file coverage

## Coverage Thresholds

Current thresholds (must meet or exceed):
- Branches: 70%
- Functions: 70%
- Lines: 70%
- Statements: 70%

## Note

This directory is git-ignored and regenerated on each test run.
